# rem ベース SCSS テンプレート

Web 制作プロジェクト用の rem 単位でレイアウトを調整する SCSS テンプレートです。

## 主な特徴

- SCSS モジュール（`@use`）を使用
- JavaScript ES6 モジュール形式
- rem でレイアウトを調整

## rem でのレイアウト調整について

このテンプレートでは、ビューポート幅に基づいて動的に rem の基準値を変更することで、レスポンシブなレイアウトを実現しています。

### 仕組み

1. **基準ビューポート幅の設定**

   - モバイル: 380px
   - デスクトップ: 1440px

2. **html 要素の font-size の動的設定**

   ```scss
   html {
     font-size: calc(100vw / var(--base-vw));
   }
   ```

   - ビューポート幅を基準幅で割ることで、1rem のサイズが画面幅に比例して変化します

### メリット

- デザインの基準幅（380px、1440px）に対して rem で指定した値がそのまま比率として機能
- メディアクエリを多用せずに、スムーズなレスポンシブ対応が可能
- フォントサイズ、余白、レイアウト幅などを統一的な rem 単位で管理

### デメリット

- タブレットや大型ディスプレイなど、基準ビューポート幅から大きく外れるデバイスでは見にくくなる
- WordPress プラグインなど、px や em など異なる単位を使用する外部ライブラリが混在するとレイアウトが破綻する
- ブラウザのフォントサイズ設定が効かなくなり、アクセシビリティの観点から問題となる

## 使用方法

1. このテンプレートを複製してプロジェクトで使用
2. SCSS を CSS にコンパイル（Sass、VS Code 拡張機能、その他のビルドツール）

## コーディング規約

プロジェクトに合わせてカスタマイズしてください。
